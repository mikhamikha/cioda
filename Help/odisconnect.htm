Работа с БД: Отключение пользователя от БД

Метод odisconnect(int)

Метод отключения пользователя от базы данных Oracle

Синтаксис

odisconnect( nNumConnect )

Параметры  
   
nNumConnect Тип: целочисленный, 32 бита
  Номер соединения, полученный oconnect() или oconnecthidden()


Возвращаемое значение

Тип: Integer
0, если успешно, или номер ошибки

Пример

INT conId1;
...
    IF DllClassIsValid( hODAC ) THEN
...
          conId1 = DllClassCallMethod( hODAC, "odisconnect", conId1 );
    ENDIF
