Работа с БД: Установить запрос к БД

Метод osetstatement(string, string)

Установка запроса SQL для последующего заполнения параметрами и выполнения

Синтаксис

osetstatement( sRequest, sXML )

Параметры  
   
sRequest Тип: строковый
  Выражение запроса или
путь до узла с запросом
   
sXML Тип: строковый
  Путь до конфигурационного файла


Возвращаемое значение

Тип: Integer
Номер сохраненного запроса (>0), если успешно, или 0, если запрос не удалось сохранить

Примечание

Запрос sRequest может быть задан либо явно как выражение (sXml в этом случае должен быть пустой строкой), либо как XPATH до требуемого атрибута в Xml-файле.

Пример

см. oexecute()
