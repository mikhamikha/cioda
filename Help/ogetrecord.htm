Работа с БД: Получить строку из результатов запроса к БД

Метод ogetrecord(int, int, int)

Получить строку данных из результатов запроса к БД

Синтаксис

ogetrecord( stmtId, recOff, suff )

Параметры  
   
stmtId Тип: Целочисленный (32 бита)
  Идентификатор запроса
   
recOff Тип: Целочисленный (32 бита)
  Номер строки ( 0 - первая строка )
   
suff Тип: Целочисленный (32 бита)
  Суффикс, добавляемый к имени тэга или объекта, указанному в бинд-листе.
Число преобразуется в строку согласно указанному формату переменной бинд-листа.


Возвращаемое значение

Тип: Integer
0, если успешно, или другое значение, если операцию выполнить не удалось

Примечание

Метод выполняет чтение строки данных из выборки после выполнения oexecute() запроса к БД.
Строка разбирается в массив данных согласно именам столбцов бинд-листа, после чего производится прямая запись полученных значений в переменные Citect по именам согласно заданного в бинд-листе соответствия.
Идентификатор запроса был получен методом osetstatement().
Бинд-лист был добавлен с помощью метода oaddbindlist().
