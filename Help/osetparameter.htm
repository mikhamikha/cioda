Работа с БД: Установка значения параметра в запросе к БД

Метод osetparameter(int, string, string)

Подстановка значений параметров в сохраненный в памяти запрос к БД

Синтаксис

osetparameter( stmtId, sName, sValue )

Параметры  
   
stmtId Тип: Целочисленный (32 бита)
  Идентификатор запроса
   
sName Тип: Строковый
  Название параметра, например, :param1
   
sValue Тип: Строковый
  Значение подставляемого параметра


Возвращаемое значение

Тип: Integer
0, если успешно, или другое значение, если операцию выполнить не удалось

Примечание

Метод выполняет подстановку значения параметра в ранее сохраненный запрос к БД.
Идентификатор запроса был получен методом osetstatement().

Пример

Пример запроса с подставляемыми параметрами:

INSERT INTO sensor_reading (FIXED_SENSOR_ID, STORAGE_ID, OPER_TYPE, FILL_UP_LEVEL, DENCITY, TEMPERATURE)
VALUES (:fixId, :storId, :type_meas, :fill_up_level, :density, :temperature)

Применение см. в oexecute()
