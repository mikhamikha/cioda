Работа с БД: Освобождение выделенных под запрос ресурсов

Метод oend(int)

Освобождение выделенных ранее ресурсов под запрос по идентификатору, полученному osetstatement()

Синтаксис

oend( stmtId )

Параметры  
   
stmtId Тип: Целочисленный (32 бита)
  Идентификатор запроса


Возвращаемое значение

Тип: Integer
0, если успешно, или другое значение, если операцию выполнить не удалось

Примечание

Метод выполняет удаление ранее сохраненного в списке запросов обращения к БД, списка соответствующих ему параметров, а также всей выборки данных, полученной ogetrecord()
Идентификатор запроса был получен методом osetstatement().

Пример

DllClassCallMethod( hODAC, "oend", stmtId1 );
