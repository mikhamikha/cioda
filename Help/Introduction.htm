Введение

Данный документ описывает принципы работы с компонентом cioda, предназначенного для предоставления наиболее функционального механизма интеграции CitectSCADA и БД Oracle.

Использование данного компонента возможно из встроенного в Citect языка программирования Cicode с помощью набора функций для взаимодействия с .Net библиотеками.

Основные особенности:

  • подключение к БД без явного ввода пароля, который "подтягивается" из зашифрованного AES-алгоритмом блока;
  • использование списков связывания для автоматического обмена данными между SCADA и БД
  • логгирование информационных сообщений и сообщений об ошибках

Используемые технологии:

  • Компонент построен на платформе .Net (версия > 4.5)
  • Для доступа к данным SCADA используется библиотека Citect API (ctapi.dll)
  • Для доступа к БД используется библиотека Oracle Data Access Component http://www.oracle.com
  • Для журналирования результатов операций используется набор компонентов Serilog https://serilog.net/

Поддерживаемые версии:

CitectSCADA - начиная с 2015

Oracle - начиная с версии 10.x
