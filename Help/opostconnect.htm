Работа с БД: Дополнительная авторизация пользователя

Метод opostconnect(int, string, string)

Метод дополнительной авторизации пользователя в БД Oracle

Синтаксис

opostconnect( connectId, sRole, sPrc )

Параметры  
  
sDsn Тип: Целочисленный (32 бита)
  Номер соединения, полученный методом oconnect()
   
sRole Тип: Строковый
  Роль пользователя
   
sPrc Тип: Строковый
  Идентификатор пользователя


Возвращаемое значение

Тип: Integer
0, если успешно, или номер ошибки

Пример

см. oconnecthidden()
