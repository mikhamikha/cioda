Работа с БД: Получить количество строк в выборке результатов запроса к БД

Метод ogetrecordnumber(int)

Получить количество строк в выборке результатов запроса к БД

Синтаксис

ogetrecordnumber( stmtId )

Параметры  
   
stmtId Тип: Целочисленный (32 бита)
  Идентификатор запроса


Возвращаемое значение

Тип: Integer
Количество строк выборки

Примечание

Метод подсчитывает количество строк данных в выборке после выполнения oexecute() запроса к БД.
Идентификатор запроса был получен методом osetstatement().
