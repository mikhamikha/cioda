Работа с БД: Подключение к БД

Метод oconnect(string, string, string)

Метод подключения к базе данных Oracle

Синтаксис

oconnect( sDsn, sUser, sPwd )

Параметры  
  
sDsn Тип: строковый
  Источник данных
   
sUser Тип: строковый
  Имя пользователя
   
sPwd Тип: строковый
  Пароль


Возвращаемое значение

Тип: Integer
Номер соединения (>0), если успешно, или 0, если подключение не установлено

Примечание

Для исключения возможности компрометирования пароля вместо данного метода рекомендуется применять метод oconnecthidden()
